Getting Started with Java

History of Java

Java programming language was originally developed by Sun Microsystems which was initiated by James Gosling and released in 1995 as core component of Sun Microsystems' Java platform (Java 1.0 [J2SE]). The latest release of the Java Standard Edition is Java SE 8. With the advancement of Java and its widespread popularity, multiple configurations were built to suit various types of platforms. For example: J2EE for Enterprise Applications, J2ME for Mobile Applications. The new J2 versions were renamed as Java SE, Java EE, and Java ME respectively. Java is guaranteed to be Write Once, Run Anywhere

Variables & DataTypes

Variable is name of reserved area allocated in memory. In other words, it is a name of memory location. It is a combination of "vary + able" that means its value can be changed.

Variable Declaration


                        int age ;

Variable Initilazation


                        int age = 40;

There are three types of variables in Java:

  • Local Variables
  • Static Variables
  • Instance Variables

Local Variable

Variable can access the Inside the method


                            public void StudentAge()
                            {
                                // local variable age
                                age = 5;
                                System.out.println("Student age is : " + age);
                            }

Static Variable

Static variable in Java is variable which belongs to the class and initialized only once at the start of the execution


                            public class Demo 
                            {
                                // static variable
                                static int a;
                                static int b;
                                
                            }

Instance Variable

Instance variables are non-static variables and are declared in a class outside any method


                            class Example
                            {
                                static int a; //static variable
                                int b;        //instance variable
                            }

Data Types

Data types specify the different sizes and values that can be stored in the variable. There are two types of data types in Java:

  • Primitive data types - includes byte, short, int, long, float, double, boolean and char
  • Non-primitive data types - such as String, Arrays,Objects and Classes

Primitive DataTypes


                            int myNum = 5;               // Integer (whole number)
                            float myFloatNum = 5.99f;    // Floating point number
                            char myLetter = 'D';         // Character
                            boolean myBool = true;       // Boolean

Operators

Operator in Java is a symbol which is used to perform operations

There are four main types of operators in Java:

Arthmetic
- Operators are +,-,*,/,%,++,--
Comparion
- Operators are ==,>,<,!=,>=,<=
Assignment
- Operators are =,+=,-=,*=,/=,%=,<<,>>
Logical
- Operators are &&(And), ||(Or), !(Not)
