Getting Started with Java

History of Java

Java programming language was originally developed by Sun Microsystems which was initiated by James Gosling and released in 1995 as core component of Sun Microsystems' Java platform (Java 1.0 [J2SE]). The latest release of the Java Standard Edition is Java SE 8. With the advancement of Java and its widespread popularity, multiple configurations were built to suit various types of platforms. For example: J2EE for Enterprise Applications, J2ME for Mobile Applications. The new J2 versions were renamed as Java SE, Java EE, and Java ME respectively. Java is guaranteed to be Write Once, Run Anywhere

Variables & DataTypes

Variable is name of reserved area allocated in memory. In other words, it is a name of memory location. It is a combination of "vary + able" that means its value can be changed.

Variable Declaration

Variable Initilazation

There are three types of variables in Java:

  • Local Variables
  • Static Variables
  • Instance Variables

Local Variable

Variable can access the Inside the method

                                age = 5;
                                System.out.println("Student age is : " + age);

Static Variable

Static variable in Java is variable which belongs to the class and initialized only once at the start of the execution

                            {

Instance Variable

Instance variables are non-static variables and are declared in a class outside any method


                            class Example

                                static int a; //static variable

Switch Statement

  • The switch expression is evaluated once
  • The value of the expression is compared with the values of each case.
  • If there is a match, the associated block of code is executed.

  • For-Each Loop
  • While Loop

For-Loop

When you know exactly how many times you want to loop through a block of code, use the for loop

                                {

For-Each

There is also a "for-each" loop, which is used exclusively to loop through elements in an array.

                                  }

While Loop

  • The while loop loops through a block of code as long as a specified condition is true:

                                Initilazation

                                {
                                    // code block to be executed
                                    Increment
                                }

Do-While Loop

  • The do/while loop is a variant of the while loop. This loop will execute the code block once, before checking if the condition is true, then it will repeat the loop as long as the condition is true.

String Methods

Strings in Java are Objects that are backed internally by a char array. Since arrays are immutable(cannot grow), Strings are immutable as well. Whenever a change to a String is made, an entirely new String is created.

The String class has a set of built-in methods that you can use on strings.

equals()

Compares two strings. Returns true if the strings are equal, and false if not

Return - Boolean

                            String myStr1 = "Hello";
                            String myStr2 = "Hello";
                            String myStr3 = "Another String";
                            System.out.println(myStr1.equals(myStr2)); // Returns true because they are equal
                            System.out.println(myStr1.equals(myStr3)); // false

length()

Returns the length of a specified string

Return - int


                            String txt = "ABCDEFGHIJKLMNOPQRSTUVWXYZ";
                            System.out.println(txt.length());

charAt()

  • Returns the character at the specified index (position)

    • Return - char

                            String myStr = "Hello";
                            char result = myStr.charAt(0);
                            System.out.println(result);

toUpperCase()

  • Converts a string to upper case letters

    • Return - String


                            String txt = "Hello World";
                            System.out.println(txt.toUpperCase());
                            System.out.println(txt.toLowerCase());

substring()

  • Extracts the characters from a string, beginning at a specified start position, and through the specified number of character

    • Return - String


                            String txt = "Hello World";
                            String txt1 = txt.substring(3,5)

replace()

  • Searches a string for a specified value, and returns a new string where the specified values are replaced

    • Return - String


                            String myStr = "Hello";
                            System.out.println(myStr.replace('l', 'p'));

isEmpty()

  • Checks whether a string is empty or not

    • Return - Boolean


                            String myStr1 = "Hello";
                            String myStr2 = "";
                            System.out.println(myStr1.isEmpty());
                            System.out.println(myStr2.isEmpty());
